This dashboard provides an overview and trend data for citizen 311 requests. Boston residents submit 311 requests to the city via phone calls and the city's Citizens Connect app.
All data used in this dashboard is sourced from the City of Boston's open data initiative, Analyze Boston.
Disclaimer: this dashboard is a work in progress, with new functionality added periodically.
